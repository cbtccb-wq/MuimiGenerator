# Assumptions Log — 無意味機構ジェネレータ

AIが暫定判断で埋めた前提・未確認事項をここに記録する。
人間レビュー必須のものには `[REVIEW]` タグを付ける。
解決済みには `[RESOLVED]` タグと結論を記入する。

---

## フォーマット

```
### [ID] {タイトル}  [REVIEW] or [RESOLVED]

- **仮定内容:**
- **理由:**
- **影響範囲:**
- **代替案:**
- **人間確認が必要か:** Yes / No
- **記録日:** YYYY-MM-DD
- **解決日:** （解決済みの場合）
- **結論:** （解決済みの場合）
```
